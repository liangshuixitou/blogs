剑指Offer-机器人的运动范围

2021-02-26 剑指Offer medium, 广度优先搜索, 深度优先搜索 0 评论 字数统计: 595(字) 阅读时长: 3(分)

机器人的运动范围

题目描述

地上有一个m行n列的方格，从坐标 [0,0] 到坐标 [m-1,n-1] 。一个机器人从坐标 [0, 0] 的格子开始移动，它每次可以向左、右、上、下移动一格（不能移动到方格外），也不能进入行坐标和列坐标的数位之和大于k的格子。例如，当k为18时，机器人能够进入方格 [35, 37] ，因为3+5+3+7=18。但它不能进入方格 [35, 38]，因为3+5+3+8=19。请问该机器人能够到达多少个格子？

示例 1：

输入：m = 2, n = 3, k = 1

输出：3

示例 2：

输入：m = 3, n = 1, k = 0

输出：1

题解

BFS

利用广度有限搜索进行渲染即可，注意我们在搜索的过程中搜索方向可以缩减为向右和向下，而不必再向上和向左进行搜索。

DFS

该题也可以利用深度优先搜索来完成，返回值为各个方向上的值相加即可，注意DFS也可以进行方向优化
